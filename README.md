# request-system

요청 관리 시스템

## 프로젝트 소개

`request-system`은 Java/Spring Boot 기반의 요청 관리 시스템입니다.

사용자가 요청을 등록하고, 등록된 요청을 조회·수정·삭제하며, 요청의 처리 상태를 관리하는 기본 CRUD 프로젝트입니다.

이 프로젝트는 Spring Boot의 기본 웹 개발 흐름을 익히기 위해 제작되었습니다.

## 개발 목적

* Spring Boot 프로젝트 구조 이해
* Controller, DTO, Entity, Repository 흐름 학습
* 요청 등록과 조회 중심의 CRUD 구현 연습
* GitHub를 활용한 프로젝트 관리 연습
* Java/Spring 백엔드 취업 준비용 포트폴리오 기반 마련

## 주요 기능

* 요청 등록
* 요청 목록 조회
* 요청 상세 조회
* 요청 수정
* 요청 삭제
* 요청 상태 변경

## 기술 스택

* Java
* Spring Boot
* Gradle
* H2 Database
* Mustache
* Git / GitHub

## 학습 포인트

이 프로젝트에서는 사용자의 입력값이 서버로 전달되고, DTO를 거쳐 객체로 변환된 뒤 저장되는 흐름을 학습합니다.

또한 화면 요청과 서버 응답이 어떻게 연결되는지, Controller가 어떤 역할을 하는지, Repository를 통해 데이터가 어떻게 저장되는지를 단계적으로 익히는 것을 목표로 합니다.

## 프로젝트 목표

단순히 코드를 따라 치는 것이 아니라, 요청 등록부터 조회까지의 전체 흐름을 직접 구현하면서 Spring Boot 백엔드 개발의 기본 감각을 쌓는 것이 목표입니다.
